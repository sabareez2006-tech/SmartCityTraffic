About

Project Overview

01

City Zoning

Divides the city into a grid of zones, each with unique traffic characteristics — residential, commercial, industrial, suburban, and downtown.

02

Traffic Data Generation

Generates synthetic traffic data with realistic peak-hour patterns, ride requests, congestion levels, and traffic flow across all city zones.

03

Federated Learning

Trains LSTM models locally at each zone, compresses gradients, selects updates via correlation analysis, and aggregates using FedAvg.

04

Gradient Compression

Applies Top-K sparsification with residual memory to reduce communication overhead by up to 85% while maintaining model accuracy.

05

Traffic Prediction

Uses the trained federated global model to predict ride demand, congestion, and traffic flow for the next time step in each zone.

06

Robotaxi Simulation

Simulates fleet dispatching under dynamic (ML-predicted) and static (baseline) traffic, comparing service rates, wait times, and utilization.
